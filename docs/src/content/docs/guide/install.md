This guide covers the default Claude Code install flow.

## Claude Code Install

### 1. Add the marketplace

```bash
/plugin marketplace add sitapix/apple-text
```

### 2. Install the plugin

```bash
/plugin install apple-text@apple-text
```

### 3. Start using it

Use `/apple-text:ask` for broad Apple text intake.
